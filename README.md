# Suzanne Ghaleb

### Installation

# after clone app!
       run command yarn with each folder project server/client
       
```sh
$ cd todo-bertoni
$ cd server && yarn install
$ cd .. && cd client && yarn install 
$  npm install cypress -g
```

# Run Server!
```sh
$ cd todo-bertoni
$ cd server && yarn start
```

    
# Run Client!
```sh
$ cd todo-bertoni
$ cd client && yarn start
```

# Run Test Server!
```sh
$ cd todo-bertoni
$ cd server && yarn start
$ yarn test
```


# Run Test Cliente!
```sh
$ cd todo-bertoni
$ cd client && yarn start
$ yarn test
```

# Run Test end2end!
```sh
$ cd todo-bertoni
$ cd server && yarn start
$ yarn cypress:open and click in spect test todo
```
